Theorem. empty_contra
x ∈ ∅ → ⊥
Proof
01. 1 ⊢ x ∈ ∅, hypo.
02. 1 ⊢ x ∈ {x | x ∈ Univ ∧ ⊥}, eq_subst empty_set_eq 1.
03. 1 ⊢ x ∈ Univ ∧ ⊥, sep_elim 2.
04. 1 ⊢ ⊥, conj_elimr 3.
empty_contra. ⊢ x ∈ ∅ → ⊥, subj_intro 4.

Dependencies
The given proof depends on two axioms:
eq_subst, separation.
