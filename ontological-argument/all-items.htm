Rules and axioms

Natural deduction inference rules

Rule. hypo
(⊤ ∧ A ⊢ A)
Rule. conj_intro
(H1 ⊢ A) → (H2 ⊢ B) → (H1 ∧ H2 ⊢ A ∧ B)
Rule. conj_eliml
(H ⊢ A ∧ B) → (H ⊢ A)
Rule. conj_elimr
(H ⊢ A ∧ B) → (H ⊢ B)
Rule. disj_introl
(H ⊢ A) → (H ⊢ A ∨ B)
Rule. disj_intror
(H ⊢ B) → (H ⊢ A ∨ B)
Rule. disj_elim
(H1 ⊢ A ∨ B) → (H2 ∧ A ⊢ C) → (H3 ∧ B ⊢ C) → (H1 ∧ H2 ∧ H3 ⊢ C)
Rule. subj_intro
(H ∧ A ⊢ B) → (H ⊢ A → B)
Rule. subj_elim
(H1 ⊢ A → B) → (H2 ⊢ A) → (H1 ∧ H2 ⊢ B)
Rule. neg_intro
(H ∧ A ⊢ ⊥) → (H ⊢ ¬A)
Rule. neg_elim
(H1 ⊢ ¬A) → (H2 ⊢ A) → (H1 ∧ H2 ⊢ ⊥)
Rule. bij_intro
(H1 ⊢ A → B) → (H2 ⊢ B → A) → (H1 ∧ H2 ⊢ A ↔ B)
Rule. bij_eliml
(H ⊢ A ↔ B) → (H ⊢ A → B)
Rule. bij_elimr
(H ⊢ A ↔ B) → (H ⊢ B → A)
Rule. uq_intro
(nf u (H ∧ ∀x. Q x)) → (H ⊢ Q u) → (H ⊢ ∀x. Q x)
Rule. uq_elim
(H ⊢ ∀x. Q x) → (H ⊢ Q t)
Rule. ex_intro
(H ⊢ Q t) → (H ⊢ ∃x. Q x)
Rule. ex_elim
(nf u (H1 ∧ H2 ∧ B ∧ ∃x. Q x)) → (H1 ⊢ ∃x. Q x) → (H2 ∧ Q u ⊢ B) → (H1 ∧ H2 ⊢ B)
Rule. nec
(⊢ A) → (⊢ □A)

Needed axioms of ZF set theory

Declaration. element_decl
x ∈ A ↔ _
Declaration. univ_decl

Definition. empty_set_eq

Scott's variant of Gödel's axioms

Declaration. positive_decl
P = _
Definition. G_eq
G = {x | x ∈ Univ ∧ ∀A. A ∈ P → x ∈ A}
Definition. ess_equi
ess E x ↔ x ∈ E ∧ ∀A. x ∈ A → □∀y. y ∈ E → y ∈ A
Definition. NE_eq
NE = {x | x ∈ Univ ∧ ∀E. ess E x → □∃y. y ∈ E}
Axiom. axiom_i
A ∈ P ↔ ¬(compl A ∈ P)
Axiom. axiom_ii
A ∈ P → □(∀x. x ∈ A → x ∈ B) → B ∈ P
Axiom. axiom_iii
G ∈ P
Axiom. axiom_iv
A ∈ P → □(A ∈ P)
Axiom. axiom_v
NE ∈ P
