Theorem God_has_any_positive_property

Theorem. God_has_any_positive_property
A ∈ P → ∀x. x ∈ G → x ∈ A
Proof
01. 1 ⊢ A ∈ P, hypo.
02. 2 ⊢ x ∈ G, hypo.
03. 2 ⊢ x ∈ {x | x ∈ Univ ∧ ∀A. A ∈ P → x ∈ A}, eq_subst G_eq 2.
04. 2 ⊢ x ∈ Univ ∧ ∀A. A ∈ P → x ∈ A, sep_elim 3.
05. 2 ⊢ ∀A. A ∈ P → x ∈ A, conj_elimr 4.
06. 2 ⊢ A ∈ P → x ∈ A, uq_elim 5.
07. 1, 2 ⊢ x ∈ A, subj_elim 6 1.
08. 1 ⊢ x ∈ G → x ∈ A, subj_intro 7.
09. 1 ⊢ ∀x. x ∈ G → x ∈ A, uq_intro 8.
God_has_any_positive_property. ⊢
  A ∈ P → ∀x. x ∈ G → x ∈ A, subj_intro 9.

Dependencies
The given proof depends on two axioms:
eq_subst, separation.
