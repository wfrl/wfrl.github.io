Theorem. compl_elim
x ∈ compl A → x ∈ Univ ∧ ¬(x ∈ A)
Proof
01. 1 ⊢ x ∈ compl A, hypo.
02. 1 ⊢ x ∈ {x | x ∈ Univ ∧ ¬(x ∈ A)},
  eq_subst compl_eq 1, Q t ↔ x ∈ t.
03. 1 ⊢ x ∈ Univ ∧ ¬(x ∈ A), sep_elim 2.
compl_elim. ⊢ x ∈ compl A → x ∈ Univ ∧ ¬(x ∈ A),
  subj_intro 3.

Dependencies
The given proof depends on two axioms:
eq_subst, separation.
