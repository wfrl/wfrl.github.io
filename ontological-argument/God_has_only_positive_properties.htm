Theorem God_has_only_positive_properties

Theorem. God_has_only_positive_properties
x ∈ G → x ∈ A → A ∈ P
Proof
01. 1 ⊢ x ∈ G, hypo.
02. 2 ⊢ x ∈ A, hypo.
03. 3 ⊢ ¬(A ∈ P), hypo.
04. ⊢ A ∈ P ↔ ¬(compl A ∈ P), axiom_i.
05. ⊢ ¬(A ∈ P) ↔ ¬¬(compl A ∈ P), equi_cong_neg 4.
06. 3 ⊢ ¬¬(compl A ∈ P), lsubj_elim 5 3.
07. 3 ⊢ compl A ∈ P, dne 6.
08. 1 ⊢ x ∈ {x | x ∈ Univ ∧ ∀A. A ∈ P → x ∈ A}, eq_subst G_eq 1.
09. 1 ⊢ x ∈ Univ ∧ ∀A. A ∈ P → x ∈ A, sep_elim 8.
10. 1 ⊢ ∀A. A ∈ P → x ∈ A, conj_elimr 9.
11. 1 ⊢ compl A ∈ P → x ∈ compl A, uq_elim 10.
12. 1, 3 ⊢ x ∈ compl A, subj_elim 11 7.
13. 1, 3 ⊢ x ∈ Univ ∧ ¬(x ∈ A), compl_elim 12.
14. 1, 3 ⊢ ¬(x ∈ A), conj_elimr 13.
15. 1, 2, 3 ⊢ ⊥, neg_elim 14 2.
16. 1, 2 ⊢ ¬¬(A ∈ P), neg_intro 15.
17. 1, 2 ⊢ A ∈ P, dne 16.
18. 1 ⊢ x ∈ A → A ∈ P, subj_intro 17.
God_has_only_positive_properties. ⊢
  x ∈ G → x ∈ A → A ∈ P, subj_intro 18.

Dependencies
The given proof depends on five axioms:
axiom_i, efq, eq_subst, lem, separation.
