Theorem. Kii
□(A → B → C) → □A → □B → □C
Proof
01. 1 ⊢ □(A → B → C), hypo.
02. 1 ⊢ □A → □(B → C), K 1.
03. 3 ⊢ □A, hypo.
04. 1, 3 ⊢ □(B → C), subj_elim 2 3.
05. 1, 3 ⊢ □B → □C, K 4.
06. 1 ⊢ □A → □B → □C, subj_intro 5.
Kii. ⊢ □(A → B → C) → □A → □B → □C, subj_intro 6.
