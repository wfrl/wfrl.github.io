Theorem. God_exists_possibly
◇∃x. x ∈ G
Proof
01. 1 ⊢ ¬◇∃x. x ∈ G, hypo.
02. ⊢ ◇(∃x. x ∈ G) ↔ ¬□¬∃x. x ∈ G, pos_equi.
03. ⊢ ¬◇(∃x. x ∈ G) ↔ ¬¬□¬∃x. x ∈ G, equi_cong_neg 2.
04. 1 ⊢ ¬¬□¬∃x. x ∈ G, lsubj_elim 3 1.
05. 1 ⊢ □¬∃x. x ∈ G, dne 4.
06. 6 ⊢ ¬∃x. x ∈ G, hypo.
07. 6 ⊢ ∀x. ¬(x ∈ G), neg_ex 6.
08. ⊢ (¬∃x. x ∈ G) → ∀x. ¬(x ∈ G), subj_intro 7.
09. ⊢ □((¬∃x. x ∈ G) → ∀x. ¬(x ∈ G)), nec 8.
10. ⊢ □(¬∃x. x ∈ G) → □∀x. ¬(x ∈ G), K 9.
11. 1 ⊢ □∀x. ¬(x ∈ G), subj_elim 10 5.

12. 12 ⊢ ∀x. ¬(x ∈ G), hypo.
13. 12 ⊢ ¬(x ∈ G), uq_elim 12.
14. 14 ⊢ x ∈ G, hypo.
15. 12, 14 ⊢ ⊥, neg_elim 13 14.
16. 12, 14 ⊢ x ∈ B, efq 15.
17. 12 ⊢ x ∈ G → x ∈ B, subj_intro 16.
18. 12 ⊢ ∀x. x ∈ G → x ∈ B, uq_intro 17.
19. ⊢ (∀x. ¬(x ∈ G)) → (∀x. x ∈ G → x ∈ B), subj_intro 18.
20. ⊢ □((∀x. ¬(x ∈ G)) → (∀x. x ∈ G → x ∈ B)), nec 19.
21. ⊢ □(∀x. ¬(x ∈ G)) → □(∀x. x ∈ G → x ∈ B), K 20.
22. 1 ⊢ □(∀x. x ∈ G → x ∈ B), subj_elim 21 11.

23. ⊢ G ∈ P, axiom_iii.
24. 1 ⊢ B ∈ P, axiom_ii 23 22.
25. 1 ⊢ ∅ ∈ P, 24.
26. ⊢ ∅ ∈ P ↔ ¬(compl ∅ ∈ P), axiom_i.
27. 1 ⊢ ¬(compl ∅ ∈ P), lsubj_elim 26 25.
28. 1 ⊢ ¬(Univ ∈ P),
  eq_subst compl_empty_set 27, Q t ↔ ¬(t ∈ P).
29. 1 ⊢ Univ ∈ P, 24.
30. 1 ⊢ ⊥, neg_elim 28 29.
31. ⊢ ¬¬◇∃x. x ∈ G, neg_intro 30.
God_exists_possibly. ⊢ ◇∃x. x ∈ G, dne 31.
