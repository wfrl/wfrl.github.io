Power sets

Theorem. power_intro
set x → x ⊆ y → x ∈ power y
Theorem. power_elim
x ∈ power M → x ⊆ M
Theorem. power_UnivCl
power UnivCl = UnivCl
Theorem. prod_incl_in_power
A × B ⊆ power (power (A ∪ B))
Theorem. set_prod
set A → set B → set (A × B)
Theorem. sep_in_power
set X → {x | x ∈ X ∧ P x} ∈ power X
Theorem. power_empty_set
power ∅ = {∅}
Theorem. power_sg
set a → power {a} = {∅, {a}}
Theorem. sg_incl_in_power
set x → {x} ⊆ power x
Theorem. function_is_set
function f → set (dom f) → set f
Theorem. map_is_set
map f X Y → set X → set f
Theorem. map_is_set_dom_cod
map f X Y → set X → set Y → set f
