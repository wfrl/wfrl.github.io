Theorem rmul_cancel

Theorem. rmul_cancel
a ∈ ℝ → b ∈ ℝ → ¬(b = 0) → a⋅b/b = a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ ¬(b = 0), hypo.
04. ⊢ a⋅b/b = a⋅b⋅/b, rdiv_eq.
05. 1, 2, 3 ⊢ a⋅b⋅/b = a, rmul_cancel_inv 1 2 3.
06. 1, 2, 3 ⊢ a⋅b/b = a, eq_trans 4 5.
rmul_cancel. ⊢ a ∈ ℝ → b ∈ ℝ →
  ¬(b = 0) → a⋅b/b = a, subj_intro_iii 6.

Dependencies
The given proof depends on six axioms:
eq_refl, eq_subst, rinv_closed, rmul_assoc, rmul_inv, rmul_neutr.
