Theorem. neg_ex_bounded_rev
(∀x. P x → ¬Q x) → (¬∃x. P x ∧ Q x)
Proof
01. 1 ⊢ ∀x. P x → ¬Q x, hypo.
02. 2 ⊢ ∃x. P x ∧ Q x, hypo.
03. 3 ⊢ P x ∧ Q x, hypo.
04. 3 ⊢ P x, conj_eliml 3.
05. 3 ⊢ Q x, conj_elimr 3.
06. 1, 3 ⊢ ¬Q x, uq_bounded_elim 1 4.
07. 1, 3 ⊢ ⊥, neg_elim 6 5.
08. 1, 2 ⊢ ⊥, ex_elim 2 7.
09. 1 ⊢ ¬∃x. P x ∧ Q x, neg_intro 8.
neg_ex_bounded_rev. ⊢ (∀x. P x → ¬Q x) → (¬∃x. P x ∧ Q x),
  subj_intro 9.
