Theorem. rlt_compat_mulr
a ∈ ℝ → b ∈ ℝ → c ∈ ℝ → a < b → 0 < c → a⋅c < b⋅c
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ c ∈ ℝ, hypo.
04. 4 ⊢ a < b, hypo.
05. 5 ⊢ 0 < c, hypo.
06. 4 ⊢ a ≤ b, le_from_lt 4.
07. 5 ⊢ 0 ≤ c, le_from_lt 5.
08. 1, 2, 3, 4, 5 ⊢ a⋅c ≤ b⋅c,
  rle_compat_mulr 1 2 3 6 7.
09. 9 ⊢ a⋅c = b⋅c, hypo.
10. 5 ⊢ ¬(0 = c), neq_from_lt 5.
11. 5 ⊢ ¬(c = 0), neq_symm 10.
12. 1, 2, 3, 5, 9 ⊢ a = b,
  rmul_cancel_rr 1 2 3 11 9.
13. 4 ⊢ ¬(a = b), neq_from_lt 4.
14. 1, 2, 3, 4, 5, 9 ⊢ ⊥, neg_elim 13 12.
15. 1, 2, 3, 4, 5 ⊢ ¬(a⋅c = b⋅c), neg_intro 14.
16. 1, 2, 3, 4, 5 ⊢ a⋅c < b⋅c, lt_intro 8 15.
17. 1, 2, 3 ⊢ a < b → 0 < c → a⋅c < b⋅c,
  subj_intro_ii 16.
rlt_compat_mulr. ⊢ a ∈ ℝ → b ∈ ℝ → c ∈ ℝ →
  a < b → 0 < c → a⋅c < b⋅c, subj_intro_iii 17.
