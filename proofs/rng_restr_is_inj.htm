Theorem rng_restr_is_inj

Theorem. rng_restr_is_inj
inj f X Y → rng f ⊆ B → inj f X B
Proof
01. 1 ⊢ inj f X Y, hypo.
02. 2 ⊢ rng f ⊆ B, hypo.
03. 1 ⊢ map f X Y, inj_is_map 1.
04. 1, 2 ⊢ map f X B,  rng_restr_is_map 3 2.
05. 1 ⊢ ∀a. ∀b. a ∈ X → b ∈ X → app f a = app f b → a = b,
  lsubj_conj_elimr inj_equi 1.
06. 1, 2 ⊢ inj f X B, inj_intro 4 5.
rng_restr_is_inj. ⊢ inj f X Y → rng f ⊆ B → inj f X B,
  subj_intro_ii 6.
