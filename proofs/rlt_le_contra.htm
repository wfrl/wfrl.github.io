Theorem rlt_le_contra

Theorem. rlt_le_contra
a ∈ ℝ → a < 0 → 0 ≤ a → ⊥
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ a < 0, hypo.
03. 3 ⊢ 0 ≤ a, hypo.
04. 2 ⊢ a ≤ 0, le_from_lt 2.
05. ⊢ 0 ∈ ℝ, calc.
06. 1, 2, 3 ⊢ a = 0, rle_antisym 1 5 4 3.
07. 2 ⊢ ¬a = 0, neq_from_lt 2.
08. 1, 2, 3 ⊢ ⊥, neg_elim 7 6.
rlt_le_contra. ⊢ a ∈ ℝ → a < 0 → 0 ≤ a → ⊥,
  subj_intro_iii 8.

Dependencies
The given proof depends on one axiom:
rle_antisym.
