Theorem incl_trans

Theorem. incl_trans
A ⊆ B → B ⊆ C → A ⊆ C

Transitivity law of inclusion. A subclass of a subclass of some class C is also a subclass of C.

Proof idea
By definition, the statement A ⊆ C means that every element of A must also be an element of C. To show this, we consider an arbitrary element of A. Since A ⊆ B, it is also an element of B. Thus, since B ⊆ C, it is also an element of C, as claimed.

Proof sketch
Let A ⊆ B and B ⊆ C. We need to show that A ⊆ C, which according to the definition unfolds as
  ∀x. x ∈ A → x ∈ C.
To do this, we consider x as a parameter that stands for a fixed but arbitrary set. We now need to show that x ∈ C under the additional assumption that x ∈ A. From A ⊆ B and x ∈ A, we first obtain x ∈ B. From B ⊆ C and x ∈ B, we then obtain x ∈ C, which which completes the proof. q.e.d.

Proof
01. 1 ⊢ A ⊆ B, hypo.
02. 2 ⊢ B ⊆ C, hypo.
03. 3 ⊢ x ∈ A, hypo.
04. 1, 3 ⊢ x ∈ B, incl_elim 1 3.
05. 1, 2, 3 ⊢ x ∈ C, incl_elim 2 4.
06. 1, 2 ⊢ x ∈ A → x ∈ C, subj_intro 5.
07. 1, 2 ⊢ ∀x. x ∈ A → x ∈ C, uq_intro 6.
08. 1, 2 ⊢ A ⊆ C, incl_intro 7.
incl_trans. ⊢ A ⊆ B → B ⊆ C → A ⊆ C, subj_intro_ii 8.
