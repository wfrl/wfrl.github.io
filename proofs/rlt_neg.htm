Theorem rlt_neg

Theorem. rlt_neg
a ∈ ℝ → b ∈ ℝ → ¬(a < b) → b ≤ a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ ¬(a < b), hypo.
04. ⊢ a < b ↔ a ≤ b ∧ ¬a = b, lt_equi.
05. ⊢ ¬(a < b) ↔ ¬(a ≤ b ∧ ¬a = b),
  equi_cong_neg 4.
06. 3 ⊢ ¬(a ≤ b ∧ ¬a = b), lsubj_elim 5 3.
07. 3 ⊢ ¬(a ≤ b) ∨  ¬¬(a = b), neg_conj 6.
08. 8 ⊢ ¬(a ≤ b), hypo.
09. 1, 2, 8 ⊢ b ≤ a, rle_neg_le 1 2 8.
10. 10 ⊢ ¬¬(a = b), hypo.
11. 10 ⊢ a = b, dne 10.
12. 10 ⊢ b = a, eq_symm 11.
13. 1 ⊢ a ≤ a, rle_refl 1.
14. 1, 10 ⊢ b ≤ a, eq_le_trans 12 13.
15. 1, 2, 3 ⊢ b ≤ a, disj_elim 7 9 14.
rlt_neg. ⊢ a ∈ ℝ → b ∈ ℝ →
  ¬(a < b) → b ≤ a, subj_intro_iii 15.

Dependencies
The given proof depends on six axioms:
efq, eq_refl, eq_subst, lem, rle_refl, rle_total.
