Theorem. map_disjoint_dom
map f X1 Y1 → map g X2 Y2 → X1 ∩ X2 = ∅ → f ∩ g = ∅
Proof
01. 1 ⊢ map f X1 Y1, hypo.
02. 2 ⊢ map g X2 Y2, hypo.
03. 3 ⊢ X1 ∩ X2 = ∅, hypo.
04. 1 ⊢ f ⊆ X1 × Y1, map_incl_in_prod 1.
05. 2 ⊢ g ⊆ X2 × Y2, map_incl_in_prod 2.
06. 1, 2, 3 ⊢ f ∩ g = ∅, relation_disjointl 4 5 3.
map_disjoint_dom. ⊢ map f X1 Y1 → map g X2 Y2 →
  X1 ∩ X2 = ∅ → f ∩ g = ∅, subj_intro_iii 6.
