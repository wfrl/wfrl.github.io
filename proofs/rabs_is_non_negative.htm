Theorem. rabs_is_non_negative
x ∈ ℝ → 0 ≤ abs x
Proof
01. 1 ⊢ x ∈ ℝ, hypo.
02. ⊢ 0 ∈ ℝ, calc.
03. 1 ⊢ 0 ≤ x ∨ x < 0, rle_lt_tricho 2 1.
04. 4 ⊢ 0 ≤ x, hypo.
05. 1, 4 ⊢ abs x = x, rabs_eql 1 4.
06. 1, 4 ⊢ x = abs x, eq_symm 5.
07. 1, 4 ⊢ 0 ≤ abs x, le_eq_trans 4 6.
08. 8 ⊢ x < 0, hypo.
09. 8 ⊢ x ≤ 0, le_from_lt 8.
10. 1, 8 ⊢ abs x = -x, rabs_non_positive 1 9.
11. 1, 8 ⊢ -x = abs x, eq_symm 10.
12. 1, 8 ⊢ -0 ≤ -x, rle_neg_swaps 1 2 9.
13. ⊢ 0 = -0, calc.
14. 1, 8 ⊢ 0 ≤ -x, eq_le_trans 13 12.
15. 1, 8 ⊢ 0 ≤ abs x, le_eq_trans 14 11.
16. 1 ⊢ 0 ≤ abs x, disj_elim 3 7 15.
rabs_is_non_negative. ⊢ x ∈ ℝ → 0 ≤ abs x,
  subj_intro 16.
