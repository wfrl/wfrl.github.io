Theorem Banach_decomposition_theorem

Theorem. Banach_decomposition_theorem
set X → rng g ⊆ X → ∃A. ∃B. A ⊆ X ∧ img f A = B ∧ img g (Y \ B) = X \ A

Given sets X, Y and two functions f: X → Y and g: Y → X. Then there exists a decomposition of X into two disjoint subsets A and X\A, and a decomposition of Y into two disjoint subsets B and Y\B, such that f[A] = B and g[Y\B] = X\A.

Proof idea
The idea is to reduce the problem to finding a suitable function on the power set of X, to which the Knaster-Tarski fixed-point theorem is applied. The second equation takes on the role of the transformed fixed-point equation, while the first equation holds by definition.

Proof sketch
We consider the power set P(X), which forms a complete lattice under the inclusion relation ⊆. We define the function
  h: P(X) → P(X), h(S) := X\g[Y\f[S]].
A fixed point of this function is a set A ⊆ X for which h(A) = A. Unfolding h, this equality is X\g[Y\f[A]] = A, which, by taking the complement of both sides, is precisely g[Y\f[A]] = X\A, the statement to be proven. To apply the Knaster-Tarski theorem, we must show that h is monotone (order-preserving), i.e., for S ⊆ T, it follows that h(S) ⊆ h(T). Let S, T ⊆ X with S ⊆ T. Then
  f[S] ⊆ f[T] (the image under a function preserves inclusion),
  Y\f[T] ⊆ Y\f[S] (taking the complement reverses inclusion),
  g[Y\f[T]] ⊆ g[Y\f[S]] (the image under a function preserves inclusion),
  X\g[Y\f[S]] ⊆ X\g[Y\f[T]] (taking the complement again reverses the inclusion).
By definition, this is equivalent to h(S) ⊆ h(T). Thus, the function h is monotone. Since h is a monotone function on the complete lattice (P(X), ⊆), the Knaster-Tarski fixed-point theorem guarantees the existence of (at least) one fixed point A ∈ P(X) such that h(A) = A. As shown, this is equivalent to the statement g[Y\B] = X\A with B := f[A]. q.e.d.
