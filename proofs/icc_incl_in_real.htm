Theorem icc_incl_in_real

Theorem. icc_incl_in_real
icc a b ⊆ ℝ
Proof
01. 1 ⊢ x ∈ icc a b, hypo.
02. 1 ⊢ x ∈ {x | x ∈ ℝ ∧ a ≤ x ∧ x ≤ b},
  eq_subst icc_eq 1, P t ↔ x ∈ t.
03. 1 ⊢ x ∈ ℝ ∧ a ≤ x ∧ x ≤ b, comp_elim 2.
04. 1 ⊢ x ∈ ℝ, conj_elimll 3.
05. ⊢ x ∈ icc a b → x ∈ ℝ, subj_intro 4.
06. ⊢ ∀x. x ∈ icc a b → x ∈ ℝ, uq_intro 5.
icc_incl_in_real. ⊢ icc a b ⊆ ℝ, incl_intro 6.

Dependencies
The given proof depends on two axioms:
comp, eq_subst.
