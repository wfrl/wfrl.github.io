Definition converges_equi

Definition. converges_equi
converges a L ↔ ∀ε. ε ∈ ℝ → 0 < ε → ∃n0. n0 ∈ ℕ ∧ ∀n. n ∈ ℕ → n0 ≤ n → abs(app a n - L) < ε

A sequence a: ℕ → ℝ is said to converge to the limit L if, for every positive ε, there exists an index n0 from which onwards all sequence values lie within the ε-neighborhood of L.
