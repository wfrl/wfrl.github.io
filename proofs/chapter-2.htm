Inference rules

Rule. hypo
(⊤ ∧ A ⊢ A)
Rule. conj_intro
(H1 ⊢ A) → (H2 ⊢ B) → (H1 ∧ H2 ⊢ A ∧ B)
Rule. conj_eliml
(H ⊢ A ∧ B) → (H ⊢ A)
Rule. conj_elimr
(H ⊢ A ∧ B) → (H ⊢ B)
Rule. disj_introl
(H ⊢ A) → (H ⊢ A ∨ B)
Rule. disj_intror
(H ⊢ B) → (H ⊢ A ∨ B)
Rule. disj_elim
(H1 ⊢ A ∨ B) → (H2 ∧ A ⊢ C) → (H3 ∧ B ⊢ C) → (H1 ∧ H2 ∧ H3 ⊢ C)
Rule. subj_intro
(H ∧ A ⊢ B) → (H ⊢ A → B)
Rule. subj_elim
(H1 ⊢ A → B) → (H2 ⊢ A) → (H1 ∧ H2 ⊢ B)
Rule. neg_intro
(H ∧ A ⊢ ⊥) → (H ⊢ ¬A)
Rule. neg_elim
(H1 ⊢ ¬A) → (H2 ⊢ A) → (H1 ∧ H2 ⊢ ⊥)
Rule. bij_intro
(H1 ⊢ A → B) → (H2 ⊢ B → A) → (H1 ∧ H2 ⊢ A ↔ B)
Rule. bij_eliml
(H ⊢ A ↔ B) → (H ⊢ A → B)
Rule. bij_elimr
(H ⊢ A ↔ B) → (H ⊢ B → A)
Rule. wk
(H ⊢ B) → (H ∧ A ⊢ B)
Rule. exch
(H ⊢ A) → (H ⊢ A)
Rule. uq_intro
(nf u (H ∧ ∀x. P x)) → (H ⊢ P u) → (H ⊢ ∀x. P x)
Rule. uq_elim
(H ⊢ ∀x. P x) → (H ⊢ P t)
Rule. ex_intro
(H ⊢ P t) → (H ⊢ ∃x. P x)
Rule. ex_elim
(nf u (H1 ∧ H2 ∧ B ∧ ∃x. P x)) → (H1 ⊢ ∃x. P x) → (H2 ∧ P u ⊢ B) → (H1 ∧ H2 ⊢ B)
Rule. subj_intro_ii
(H ∧ A ∧ B ⊢ C) → (H ⊢ A → B → C)
Rule. subj_intro_iii
(H ∧ A ∧ B ∧ C ⊢ D) → (H ⊢ A → B → C → D)
Rule. subj_intro_iv
(H ∧ A ∧ B ∧ C ∧ D ⊢ E) → (H ⊢ A → B → C → D → E)
