Theorem. ico_elim
x ∈ ico a b → x ∈ ℝ ∧ a ≤ x ∧ x < b
Proof
01. 1 ⊢ x ∈ ico a b, hypo.
02. 1 ⊢ x ∈ {x | x ∈ ℝ ∧ a ≤ x ∧ x < b},
  eq_subst ico_eq 1, P t ↔ x ∈ t.
03. 1 ⊢ x ∈ ℝ ∧ a ≤ x ∧ x < b, comp_elim 2.
ico_elim. ⊢ x ∈ ico a b → x ∈ ℝ ∧ a ≤ x ∧ x < b,
  subj_intro 3.

Dependencies
The given proof depends on two axioms:
comp, eq_subst.
