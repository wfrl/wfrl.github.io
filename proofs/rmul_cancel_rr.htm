Theorem rmul_cancel_rr

Theorem. rmul_cancel_rr
a ∈ ℝ → b ∈ ℝ → c ∈ ℝ → ¬(c = 0) → a⋅c = b⋅c → a = b
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ c ∈ ℝ, hypo.
04. 4 ⊢ ¬(c = 0), hypo.
05. 5 ⊢ a⋅c = b⋅c, hypo.
06. 5 ⊢ a⋅c⋅/c = b⋅c⋅/c, eq_cong 5, f t = t⋅/c.
07. 3, 4 ⊢ /c ∈ ℝ, rinv_closed 3 4.
08. 1, 3, 4 ⊢ a⋅c⋅/c = a⋅(c⋅/c), rmul_assoc 1 3 7.
09. 2, 3, 4 ⊢ b⋅c⋅/c = b⋅(c⋅/c), rmul_assoc 2 3 7.
10. 1, 3, 4, 5 ⊢ a⋅(c⋅/c) = b⋅c⋅/c, eq_trans_ll 8 6.
11. 1, 2, 3, 4, 5 ⊢ a⋅(c⋅/c) = b⋅(c⋅/c), eq_trans 10 9.
12. 3, 4 ⊢ c⋅/c = 1, rmul_inv 3 4.
13. 1, 2, 3, 4, 5 ⊢ a⋅1 = b⋅1,
  eq_subst 12 11, P t ↔ a⋅t = b⋅t.
14. 1 ⊢ a⋅1 = a, rmul_neutr 1.
15. 2 ⊢ b⋅1 = b, rmul_neutr 2.
16. 1, 2, 3, 4, 5 ⊢ a = b⋅1, eq_trans_ll 14 13.
17. 1, 2, 3, 4, 5 ⊢ a = b, eq_trans 16 15.
18. 1, 2, 3 ⊢ ¬(c = 0) → a⋅c = b⋅c → a = b,
  subj_intro_ii 17.
rmul_cancel_rr. ⊢ a ∈ ℝ → b ∈ ℝ → c ∈ ℝ →
   ¬(c = 0) → a⋅c = b⋅c → a = b, subj_intro_iii 18.

Dependencies
The given proof depends on six axioms:
eq_refl, eq_subst, rinv_closed, rmul_assoc, rmul_inv, rmul_neutr.
