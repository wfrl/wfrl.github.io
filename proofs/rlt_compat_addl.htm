Theorem rlt_compat_addl

Theorem. rlt_compat_addl
a ∈ ℝ → b ∈ ℝ → c ∈ ℝ → a < b → c + a < c + b
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ c ∈ ℝ, hypo.
04. 4 ⊢ a < b, hypo.
05. 1, 2, 3, 4 ⊢ a + c < b + c, rlt_compat_add 1 2 3 4.
06. 1, 3 ⊢ c + a = a + c, radd_comm 3 1.
07. 2, 3 ⊢ b + c = c + b, radd_comm 2 3.
08. 1, 2, 3, 4 ⊢ c + a < b + c, eq_lt_trans 6 5.
09. 1, 2, 3, 4 ⊢ c + a < c + b, lt_eq_trans 8 7.
rlt_compat_addl. ⊢ a ∈ ℝ → b ∈ ℝ → c ∈ ℝ →
  a < b → c + a < c + b, subj_intro_iv 9.

Dependencies
The given proof depends on nine axioms:
eq_refl, eq_subst, radd_assoc, radd_closed, radd_comm, radd_inv, radd_neutr, rle_compat_add, rneg_closed.
