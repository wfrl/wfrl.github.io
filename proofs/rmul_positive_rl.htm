Theorem. rmul_positive_rl
a ∈ ℝ → b ∈ ℝ → 0 < b → 0 < a⋅b → 0 < a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ 0 < b, hypo.
04. 4 ⊢ 0 < a⋅b, hypo.
05. 1, 2 ⊢ a⋅b = b⋅a, rmul_comm 1 2.
06. 1, 2, 4 ⊢ 0 < b⋅a, lt_eq_trans 4 5.
07. 1, 2, 3, 4 ⊢ 0 < a, rmul_positive_lr 2 1 3 6.
rmul_positive_rl. ⊢ a ∈ ℝ → b ∈ ℝ →
  0 < b → 0 < a⋅b → 0 < a, subj_intro_iv 7.
