Theorem. rdiv_positive
a ∈ ℝ → b ∈ ℝ → 0 < a → 0 < b → 0 < a/b
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ 0 < a, hypo.
04. 4 ⊢ 0 < b, hypo.
05. 2, 4 ⊢ /b ∈ ℝ, rinv_closed_positive 2 4.
06. 2, 4 ⊢ 0 < /b, rinv_positive 2 4.
07. 1, 2, 3, 4 ⊢ 0 < a⋅/b, rlt_compat_mul 1 5 3 6.
08. ⊢ a/b = a⋅/b, rdiv_eq.
09. ⊢ a⋅/b = a/b, eq_symm 8.
10. 1, 2, 3, 4 ⊢ 0 < a/b, lt_eq_trans 7 9.
rdiv_positive. ⊢ a ∈ ℝ → b ∈ ℝ →
  0 < a → 0 < b → 0 < a/b, subj_intro_iv 10.
