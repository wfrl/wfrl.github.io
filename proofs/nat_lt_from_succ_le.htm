Theorem. nat_lt_from_succ_le
n ∈ ℕ → k ∈ ℕ → n + 1 ≤ k → n < k
Proof
01. 1 ⊢ n ∈ ℕ, hypo.
02. 2 ⊢ k ∈ ℕ, hypo.
03. 3 ⊢ n + 1 ≤ k, hypo.
04. 1 ⊢ n < n + 1, nlt_succ 1.
05. 1 ⊢ n + 1 ∈ ℕ, succ_in_nat 1.
06. 1, 2, 3 ⊢ n < k, nlt_le_trans 1 5 2 4 3.
nat_lt_from_succ_le. ⊢ n ∈ ℕ → k ∈ ℕ → n + 1 ≤ k → n < k,
  subj_intro_iii 6.
