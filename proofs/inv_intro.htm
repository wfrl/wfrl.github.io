Theorem. inv_intro
(x, y) ∈ R → (y, x) ∈ inv R
Proof
01. 1 ⊢ (x, y) ∈ R, hypo.
02. ⊢ (y, x) = (y, x), eq_refl.
03. 1 ⊢ (y, x) = (y, x) ∧ (x, y) ∈ R, conj_intro 2 1.
04. 1 ⊢ ∃b. (y, x) = (b, x) ∧ (x, b) ∈ R, ex_intro 3.
05. 1 ⊢ ∃a. ∃b. (y, x) = (b, a) ∧ (a, b) ∈ R, ex_intro 4.
06. 1 ⊢ set (x, y), set_intro 1.
07. 1 ⊢ set x, setl_from_pair 6.
08. 1 ⊢ set y, setr_from_pair 6.
09. 1 ⊢ set (y, x), set_pair 8 7.
10. 1 ⊢ (y, x) ∈ {t | ∃a. ∃b. t = (b, a) ∧ (a, b) ∈ R},
  comp_intro 9 5, P t ↔ (∃a. ∃b. t = (b, a) ∧ (a, b) ∈ R).
11. 1 ⊢ (y, x) ∈ inv R,
  eq_subst_rev inv_eq 10, P A ↔ (y, x) ∈ A.
inv_intro. ⊢ (x, y) ∈ R → (y, x) ∈ inv R, subj_intro 11.
