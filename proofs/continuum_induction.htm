Theorem continuum_induction

Theorem. continuum_induction
a ∈ ℝ → b ∈ ℝ → a < b → S ⊆ ico a b → (∀x. x ∈ ico a b → ico a x ⊆ S → ∃y. y ∈ ℝ ∧ x < y ∧ ico a y ⊆ S) → S = ico a b

Induction principle for the continuum of real numbers.

Let a < b and S ⊆ [a, b). If
  ∀x ∈ [a, b). [a, x) ⊆ S → ∃y > x. [a, y) ⊆ S
then S = [a, b). The base case is contained via x := a, since [a, a) = ∅.

Proof sketch
Let A := {x ∈ [a, b] | [a, x) ⊆ S}.

1. Show that A is non-empty and bounded above. Non-empty: Since a ∈ [a, b) and [a, a) = ∅ ⊆ S, our assumption for x = a implies there is a y > a with [a, y) ⊆ S. Since S ⊆ [a, b), we have [a, y) ⊆ [a, b), which means y ≤ b. Thus, y ∈ [a, b], and by definition y ∈ A. Therefore, A ≠ ∅. Bounded above: By definition, if x ∈ A, then x ∈ [a, b], so x ≤ b. Thus, b is an upper bound for A.

2. Define c as the supremum of A. Since A is a non-empty, bounded-above subset of ℝ, by the least-upper-bound property, it has a supremum. Let c := sup A.

3. Show that [a, c) ⊆ S. Let p ∈ [a, c). Then a ≤ p < c. Since c = sup A, p is not an upper bound for A. Thus, there exists an x ∈ A with p < x. By definition of A, we have [a, x) ⊆ S. Since a ≤ p < x, it follows that p ∈ [a, x), and therefore p ∈ S. This proves [a, c) ⊆ S.

4. Show that c = b. Since b is an upper bound for A and c is the least upper bound, we have c ≤ b. Assume for contradiction that c < b. Since a ∈ A, we have a ≤ c, which means c ∈ [a, b). From step 3, we know [a, c) ⊆ S. By our main assumption (with x = c), there must exist a y ∈ ℝ such that c < y and [a, y) ⊆ S. From [a, y) ⊆ S ⊆ [a, b), we deduce y ≤ b. Thus, y ∈ [a, b] and [a, y) ⊆ S, which means y ∈ A. But y ∈ A implies y ≤ c (since c = sup A). This contradicts c < y. The assumption c < b must be false. Hence, c ≥ b. Combining c ≤ b and c ≥ b, we conclude c = b.

5. Conclusion. Substituting c = b into the result from step 3, we get [a, b) ⊆ S. With the given hypothesis S ⊆ [a,  b), we have S = [a, b). q.e.d.
