Theorem. bij_app_inv_app
bij f X Y → x ∈ X → app (inv f) (app f x) = x
Proof
01. 1 ⊢ bij f X Y, hypo.
02. 2 ⊢ x ∈ X, hypo.
03. 1 ⊢ map f X Y, bij_is_map 1.
04. 1 ⊢ map (inv f) Y X, inv_of_bij_is_map 1.
05. ⊢ app f x = app f x, eq_refl.
06. 1, 2 ⊢ (x, app f x) ∈ f, map_app_elim 3 2 5.
07. 1, 2 ⊢ (app f x, x) ∈ inv f, inv_intro 6.
08. 1, 2 ⊢ x = app (inv f) (app f x), map_app_intro 4 7.
09. 1, 2 ⊢ app (inv f) (app f x) = x, eq_symm 8.
bij_app_inv_app. ⊢ bij f X Y → x ∈ X →
  app (inv f) (app f x) = x, subj_intro_ii 9.

Dependencies
The given proof depends on nine axioms:
comp, efq, eq_refl, eq_subst, ext, lem, pairing, subset, union.
