Theorem. sur_union
sur f X1 Y1 → sur g X2 Y2 → X1 ∩ X2 = ∅ → sur (f ∪ g) (X1 ∪ X2) (Y1 ∪ Y2)
Proof
01. 1 ⊢ sur f X1 Y1, hypo.
02. 2 ⊢ sur g X2 Y2, hypo.
03. 3 ⊢ X1 ∩ X2 = ∅, hypo.
04. 1 ⊢ map f X1 Y1, sur_is_map 1.
05. 2 ⊢ map g X2 Y2, sur_is_map 2.
06. 1, 2, 3 ⊢ map (f ∪ g) (X1 ∪ X2) (Y1 ∪ Y2),
  map_union 4 5 3.
07. ⊢ rng (f ∪ g) = rng f ∪ rng g, rng_union.
08. 1 ⊢ rng f = Y1, sur_elim 1.
09. 2 ⊢ rng g = Y2, sur_elim 2.
10. 1, 2 ⊢ rng f ∪ rng g = Y1 ∪ Y2,
  eq_cong_ii 8 9, f x y = x ∪ y.
11. 1, 2 ⊢ rng (f ∪ g) = Y1 ∪ Y2, eq_trans 7 10.
12. 1, 2 ⊢ Y1 ∪ Y2 ⊆ rng (f ∪ g), incl_from_eq_rev 11.
13. 1, 2, 3 ⊢ sur (f ∪ g) (X1 ∪ X2) (Y1 ∪ Y2), sur_intro 6 12.
sur_union. ⊢ sur f X1 Y1 → sur g X2 Y2 →
  X1 ∩ X2 = ∅ → sur (f ∪ g) (X1 ∪ X2) (Y1 ∪ Y2),
  subj_intro_iii 13.

Dependencies
The given proof depends on seven axioms:
comp, efq, eq_refl, eq_subst, ext, lem, subset.
