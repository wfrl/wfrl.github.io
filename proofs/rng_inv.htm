Theorem. rng_inv
rng (inv R) = dom R
Proof
01. 1 ⊢ x ∈ rng (inv R), hypo.
02. 1 ⊢ ∃y. (y, x) ∈ inv R, rng_elim 1.
03. 3 ⊢ (y, x) ∈ inv R, hypo.
04. 3 ⊢ (x, y) ∈ R, inv_elim 3.
05. 3 ⊢ x ∈ dom R, dom_intro 4.
06. 1 ⊢ x ∈ dom R, ex_elim 2 5.
07. ⊢ x ∈ rng (inv R) → x ∈ dom R, subj_intro 6.
08. 8 ⊢ x ∈ dom R, hypo.
09. 8 ⊢ ∃y. (x, y) ∈ R, dom_elim 8.
10. 10 ⊢ (x, y) ∈ R, hypo.
11. 10 ⊢ (y, x) ∈ inv R, inv_intro 10.
12. 10 ⊢ x ∈ rng (inv R), rng_intro 11.
13. 8 ⊢ x ∈ rng (inv R), ex_elim 9 12.
14. ⊢ x ∈ dom R → x ∈ rng (inv R), subj_intro 13.
15. ⊢ x ∈ rng (inv R) ↔ x ∈ dom R, bij_intro 7 14.
16. ⊢ ∀x. x ∈ rng (inv R) ↔ x ∈ dom R, uq_intro 15.
rng_inv. ⊢ rng (inv R) = dom R, ext 16.
