Theorem rdiv_closed_positive

Theorem. rdiv_closed_positive
a ∈ ℝ → b ∈ ℝ → 0 < b → a/b ∈ ℝ
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ 0 < b, hypo.
04. 3 ⊢ ¬(0 = b), neq_from_lt 3.
05. 3 ⊢ ¬(b = 0), neq_symm 4.
06. 1, 2, 3 ⊢ a/b ∈ ℝ, rdiv_closed 1 2 5.
rdiv_closed_positive. ⊢ a ∈ ℝ → b ∈ ℝ →
  0 < b → a/b ∈ ℝ, subj_intro_iii 6.

Dependencies
The given proof depends on four axioms:
eq_refl, eq_subst, rinv_closed, rmul_closed.
