Theorem rle_neg

Theorem. rle_neg
a ∈ ℝ → b ∈ ℝ → ¬(a ≤ b) → b < a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ ¬a ≤ b, hypo.
04. 1, 2, 3 ⊢ b ≤ a, rle_neg_le 1 2 3.
05. 5 ⊢ b = a, hypo.
06. 5 ⊢ a = b, eq_symm 5.
07. 1 ⊢ a ≤ a, rle_refl 1.
08. 1, 5 ⊢ a ≤ b, le_eq_trans 7 6.
09. 1, 3, 5 ⊢ ⊥, neg_elim 3 8.
10. 1, 3 ⊢ ¬b = a, neg_intro 9.
11. 1, 2, 3 ⊢ b < a, lt_intro 4 10.
rle_neg. ⊢ a ∈ ℝ → b ∈ ℝ →
  ¬(a ≤ b) → b < a, subj_intro_iii 11.

Dependencies
The given proof depends on five axioms:
efq, eq_refl, eq_subst, rle_refl, rle_total.
