Theorem rng_restr_is_sur

Theorem. rng_restr_is_sur
map f X Y → rng f = B → sur f X B
Proof
01. 1 ⊢ map f X Y, hypo.
02. 2 ⊢ rng f = B, hypo.
03. 2 ⊢ rng f ⊆ B, incl_from_eq 2.
04. 1, 2 ⊢ map f X B, rng_restr_is_map 1 3.
05. 2 ⊢ B ⊆ rng f, incl_from_eq_rev 2.
06. 1, 2 ⊢ sur f X B, sur_intro 4 5.
rng_restr_is_sur. ⊢ map f X Y → rng f = B →
  sur f X B, subj_intro_ii 6.

Dependencies
The given proof depends on two axioms:
eq_subst, ext.
