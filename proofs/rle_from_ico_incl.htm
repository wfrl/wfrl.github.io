Theorem rle_from_ico_incl

Theorem. rle_from_ico_incl
x ∈ ℝ → y ∈ ℝ → a ≤ y → ico a x ⊆ ico a y → x ≤ y
Proof
01. 1 ⊢ x ∈ ℝ, hypo.
02. 2 ⊢ y ∈ ℝ, hypo.
03. 3 ⊢ a ≤ y, hypo.
04. 4 ⊢ ico a x ⊆ ico a y, hypo.
05. 5 ⊢ y < x, hypo.
06. 2, 3, 5 ⊢ y ∈ ico a x, ico_intro 2 3 5.
07. 2, 3, 4, 5 ⊢ y ∈ ico a y, incl_elim 4 6.
08. 2, 3, 4, 5 ⊢ y ∈ ℝ ∧ a ≤ y ∧ y < y, ico_elim 7.
09. 2, 3, 4, 5 ⊢ y < y, conj_elimr 8.
10. 2, 3, 4, 5 ⊢ ⊥, lt_self_contra 9.
11. 2, 3, 4 ⊢ ¬(y < x), neg_intro 10.
12. 1, 2, 3, 4 ⊢ x ≤ y, rlt_neg 2 1 11.
rle_from_ico_incl. ⊢ x ∈ ℝ → y ∈ ℝ → a ≤ y →
  ico a x ⊆ ico a y → x ≤ y, subj_intro_iv 12.

Dependencies
The given proof depends on seven axioms:
comp, efq, eq_refl, eq_subst, lem, rle_refl, rle_total.
