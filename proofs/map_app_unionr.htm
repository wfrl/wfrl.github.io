Theorem. map_app_unionr
map g X2 Y → map (f ∪ g) (X1 ∪ X2) Z → x ∈ X2 → app (f ∪ g) x = app g x
Proof
01. 1 ⊢ map g X2 Y, hypo.
02. 2 ⊢ map (f ∪ g) (X1 ∪ X2) Z, hypo.
03. 3 ⊢ x ∈ X2, hypo.
04. ⊢ app g x = app g x, eq_refl.
05. 1, 3 ⊢ (x, app g x) ∈ g, map_app_elim 1 3 4.
06. 1, 3 ⊢ (x, app g x) ∈ f ∪ g, union_intror 5.
07. 1, 2, 3 ⊢ app g x = app (f ∪ g) x, map_app_intro 2 6.
08. 1, 2, 3 ⊢ app (f ∪ g) x = app g x, eq_symm 7.
map_app_unionr. ⊢ map g X2 Y → map (f ∪ g) (X1 ∪ X2) Z →
  x ∈ X2 → app (f ∪ g) x = app g x, subj_intro_iii 8.

Dependencies
The given proof depends on seven axioms:
comp, efq, eq_refl, eq_subst, ext, lem, subset.
