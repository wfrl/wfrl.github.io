Theorem intersection_incl_in_intersection

Theorem. intersection_incl_in_intersection
A1 ⊆ B1 → A2 ⊆ B2 → A1 ∩ A2 ⊆ B1 ∩ B2
Proof
01. 1 ⊢ A1 ⊆ B1, hypo.
02. 2 ⊢ A2 ⊆ B2, hypo.
03. 3 ⊢ x ∈ A1 ∩ A2, hypo.
04. 3 ⊢ x ∈ A1, intersection_eliml 3.
05. 3 ⊢ x ∈ A2, intersection_elimr 3.
06. 1, 3 ⊢ x ∈ B1, incl_elim 1 4.
07. 2, 3 ⊢ x ∈ B2, incl_elim 2 5.
08. 1, 2, 3 ⊢ x ∈ B1 ∩ B2, intersection_intro 6 7.
09. 1, 2 ⊢ x ∈ A1 ∩ A2 → x ∈ B1 ∩ B2, subj_intro 8.
10. 1, 2 ⊢ ∀x. x ∈ A1 ∩ A2 → x ∈ B1 ∩ B2, uq_intro 9.
11. 1, 2 ⊢ A1 ∩ A2 ⊆ B1 ∩ B2, incl_intro 10.
intersection_incl_in_intersection. ⊢ A1 ⊆ B1 → A2 ⊆ B2 →
  A1 ∩ A2 ⊆ B1 ∩ B2, subj_intro_ii 11.

Dependencies
The given proof depends on three axioms:
comp, eq_refl, eq_subst.
