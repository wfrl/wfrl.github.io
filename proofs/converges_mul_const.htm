Theorem converges_mul_const

Theorem. converges_mul_const
r ∈ ℝ → L ∈ ℝ → map a ℕ ℝ → converges a L → (∀n. n ∈ ℕ → app b n = r⋅(app a n)) → converges b (r⋅L)

This theorem states that a sequence scaled by a constant factor r converges to r⋅L if the original one converges to L.

Proof idea
Scaling the sequence values by the constant factor r can be interpreted as a benign coordinate transformation of the value space, which of course should not change the convergence behavior of the sequence. Accordingly, an ε-neighborhood of the scaled sequence corresponds exactly to the neighborhood with scaled radius ε/|r| in the original sequence. Since we know that it dives in and remains there, this must also be the case for the scaled sequence. This line of thought should be reflected in the fact that the inequality remains valid under changed perspective, even though it takes on a different form.

Proof sketch
Let (an) converge to L. We need to show that (r⋅an) converges to r⋅L. Case distinction between r = 0 and r ≠ 0. In the case r = 0, (r⋅an) is a constant sequence that converges to zero according to const_converges. Since r⋅L = 0, the limit coincides with r⋅L as claimed. We may now assume r ≠ 0. Let ε > 0 be fixed, but arbitrary. Let ε' := ε/|r|. Since (an) converges, there must exist an n0 such that |an − L| < ε' for every n ≥ n0. Since |r| > 0, this inequality can be equivalently transformed to
  |r⋅an − r⋅L| = |r|⋅|an − L| < ε.
Thus, for every ε, there exists an index n0 from which on this inequality is satisfied, meaning that (r⋅an) must converge to r⋅L. q.e.d.
