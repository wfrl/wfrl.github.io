Theorem Archimedean_property_corollary_iii

Theorem. Archimedean_property_corollary_iii
a ∈ ℝ → 0 ≤ a → (∀n. n ∈ ℕ ∧ 0 < n → a ≤ 1/n) → a = 0

Proof idea
In essence, this is merely the contraposition of corollary ii.

Proof sketch
Let a ∈ ℝ with 0 ≤ a be arbritary. We specialise corollary ii with ε := a to obtain
  0 < a → ∃n ∈ ℕ. 0 < n ∧ 1/n < a.
So in contraposition,
  (∀n ∈ ℕ. 0 < n → a ≤ 1/n) → a ≤ 0.
And since 0 ≤ a, the conclusion is equivalent to a = 0. q.e.d.
