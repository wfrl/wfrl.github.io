Theorem. ico_intro
x ∈ ℝ → a ≤ x → x < b → x ∈ ico a b
Proof
01. 1 ⊢ x ∈ ℝ, hypo.
02. 2 ⊢ a ≤ x, hypo.
03. 3 ⊢ x < b, hypo.
04. 1, 2 ⊢ x ∈ ℝ ∧ a ≤ x, conj_intro 1 2.
05. 1, 2, 3 ⊢ x ∈ ℝ ∧ a ≤ x ∧ x < b, conj_intro 4 3.
06. 1 ⊢ set x, set_intro 1.
07. 1, 2, 3 ⊢ x ∈ {x | x ∈ ℝ ∧ a ≤ x ∧ x < b}, comp_intro 6 5.
08. 1, 2, 3 ⊢ x ∈ ico a b, eq_subst_rev ico_eq 7, P t ↔ x ∈ t.
ico_intro. ⊢ x ∈ ℝ → a ≤ x → x < b → x ∈ ico a b,
  subj_intro_iii 8.

Dependencies
The given proof depends on three axioms:
comp, eq_refl, eq_subst.
