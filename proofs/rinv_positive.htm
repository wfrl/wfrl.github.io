Theorem. rinv_positive
a ∈ ℝ → 0 < a → 0 < /a

05. ⊢ 0 ∈ ℝ, calc.
06. 2 ⊢ ¬(0 = a), neq_from_lt 2.

13. 1, 2, 3 ⊢ 1 ≤ a⋅0, eq_le_trans 11 9.
14. 1, 2, 3 ⊢ 1 ≤ 0, le_eq_trans 13 12.
15. ⊢ ¬(1 ≤ 0), calc.
16. 1, 2, 3 ⊢ ⊥, neg_elim 15 14.
17. 1, 2 ⊢ ¬(/a ≤ 0), neg_intro 16.
18. 1, 2 ⊢ 0 < /a, rle_neg 8 5 17.
rinv_positive. ⊢ a ∈ ℝ →
  0 < a → 0 < /a, subj_intro_ii 18.
