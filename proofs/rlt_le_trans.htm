Theorem rlt_le_trans

Theorem. rlt_le_trans
a ∈ ℝ → b ∈ ℝ → c ∈ ℝ → a < b → b ≤ c → a < c
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ c ∈ ℝ, hypo.
04. 4 ⊢ a < b, hypo.
05. 5 ⊢ b ≤ c, hypo.
06. 4 ⊢ a ≤ b, le_from_lt 4.
07. 1, 2, 3, 4, 5 ⊢ a ≤ c, rle_trans 1 2 3 6 5.
08. 8 ⊢ a = c, hypo.
09. 8 ⊢ c = a, eq_symm 8.
10. 5, 8 ⊢ b ≤ a, le_eq_trans 5 9.
11. 1, 2, 4, 5, 8 ⊢ a = b, rle_antisym 1 2 6 10.
12. 4 ⊢ ¬a = b, neq_from_lt 4.
13. 1, 2, 4, 5, 8 ⊢ ⊥, neg_elim 12 11.
14. 1, 2, 4, 5 ⊢ ¬a = c, neg_intro 13.
15. 1, 2, 3, 4, 5 ⊢ a < c, lt_intro 7 14.
16. 1, 2, 3 ⊢ a < b → b ≤ c → a < c, subj_intro_ii 15.
rlt_le_trans. ⊢ a ∈ ℝ → b ∈ ℝ → c ∈ ℝ →
  a < b → b ≤ c → a < c, subj_intro_iii 16.

Dependencies
The given proof depends on four axioms:
eq_refl, eq_subst, rle_antisym, rle_trans.
