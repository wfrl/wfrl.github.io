Theorem. map_app_unionl
map f X1 Y → map (f ∪ g) (X1 ∪ X2) Z → x ∈ X1 → app (f ∪ g) x = app f x
Proof
01. 1 ⊢ map f X1 Y, hypo.
02. 2 ⊢ map (f ∪ g) (X1 ∪ X2) Z, hypo.
03. 3 ⊢ x ∈ X1, hypo.
04. ⊢ app f x = app f x, eq_refl.
05. 1, 3 ⊢ (x, app f x) ∈ f, map_app_elim 1 3 4.
06. 1, 3 ⊢ (x, app f x) ∈ f ∪ g, union_introl 5.
07. 1, 2, 3 ⊢ app f x = app (f ∪ g) x, map_app_intro 2 6.
08. 1, 2, 3 ⊢ app (f ∪ g) x = app f x, eq_symm 7.
map_app_unionl. ⊢ map f X1 Y → map (f ∪ g) (X1 ∪ X2) Z →
  x ∈ X1 → app (f ∪ g) x = app f x, subj_intro_iii 8.

Dependencies
The given proof depends on seven axioms:
comp, efq, eq_refl, eq_subst, ext, lem, subset.
