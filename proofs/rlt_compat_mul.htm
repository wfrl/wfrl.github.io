Theorem. rlt_compat_mul
a ∈ ℝ → b ∈ ℝ → 0 < a → 0 < b → 0 < a⋅b
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ 0 < a, hypo.
04. 4 ⊢ 0 < b, hypo.
05. 3 ⊢ 0 ≤ a, le_from_lt 3.
06. 4 ⊢ 0 ≤ b, le_from_lt 4.
07. 1, 2, 3, 4 ⊢ 0 ≤ a⋅b, rle_compat_mul 1 2 5 6.
08. 3 ⊢ ¬(0 = a), neq_from_lt 3.
09. 4 ⊢ ¬(0 = b), neq_from_lt 4.
10. 3 ⊢ ¬(a = 0), neq_symm 8.
11. 4 ⊢ ¬(b = 0), neq_symm 9.
12. 1, 2, 3, 4 ⊢ ¬(a⋅b = 0), rmul_non_zero 1 2 10 11.
13. 1, 2, 3, 4 ⊢ ¬(0 = a⋅b), neq_symm 12.
14. 1, 2, 3, 4 ⊢ 0 < a⋅b, lt_intro 7 13.
rlt_compat_mul. ⊢ a ∈ ℝ → b ∈ ℝ →
  0 < a → 0 < b → 0 < a⋅b, subj_intro_iv 14.
