Theorem rng_restr_is_map

Theorem. rng_restr_is_map
map f X Y → rng f ⊆ B → map f X B
Proof
01. 1 ⊢ map f X Y, hypo.
02. 2 ⊢ rng f ⊆ B, hypo.
03. 1 ⊢ function f, map_is_function 1.
04. 1 ⊢ dom f = X, map_dom 1.
05. 1, 2 ⊢ map f X B, map_intro 3 4 2.
rng_restr_is_map. ⊢ map f X Y → rng f ⊆ B →
  map f X B, subj_intro_ii 5.
