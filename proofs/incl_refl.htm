Theorem incl_refl

Theorem. incl_refl
A ⊆ A

Reflexivity law of inclusion. Each class is a subclass of itself.

Proof idea
By definition, the statement A ⊆ A means that every element of A must also be an element of A, which is trivially a factual state of affairs.

Proof sketch
If we unfold the definition of inclusion, the statement A ⊆ A takes the form
  ∀x. x ∈ A → x ∈ A.
To confirm this, we consider x as a parameter that stands for a fixed but arbitrary set. With regard to this, we now have to show x ∈ A → x ∈ A. This means that, assuming x ∈ A, we have to derive x ∈ A, which is the assumption itself, so we have already reached our goal. q.e.d.

Proof
01. 1 ⊢ x ∈ A, hypo.
02. ⊢ x ∈ A → x ∈ A, subj_intro 1.
03. ⊢ ∀x. x ∈ A → x ∈ A, uq_intro 2.
incl_refl. ⊢ A ⊆ A, incl_intro 3.
