Upper bounds

Theorem. is_sup_unfold
is_sup s A → upper_bound s A ∧ (∀b. b ∈ ℝ → upper_bound b A → s ≤ b)
Theorem. upper_bound_fold
(∀x. x ∈ A → x ≤ b) → upper_bound b A
Theorem. upper_bound_elim
upper_bound b A → x ∈ A → x ≤ b
Theorem. upper_bound_neg
A ⊆ ℝ → b ∈ ℝ → ¬upper_bound b A → ∃x. x ∈ A ∧ b < x
Theorem. no_upper_bound_lt_sup
s ∈ ℝ → x ∈ ℝ → is_sup s A → x < s → ¬upper_bound x A
Theorem. upper_bound_weaken
a ∈ ℝ → b ∈ ℝ → A ⊆ ℝ → upper_bound a A → a < b → upper_bound b A

Archimedean property

Theorem. nat_unbounded
¬(∃b. b ∈ ℝ ∧ upper_bound b ℕ)
Theorem. Archimedean_property
x ∈ ℝ → ∃n. n ∈ ℕ ∧ x < n
Theorem. Archimedean_property_corollary
x ∈ ℝ → y ∈ ℝ → 0 < x → ∃n. n ∈ ℕ ∧ y < n⋅x
Theorem. Archimedean_property_corollary_ii
ε ∈ ℝ → 0 < ε → ∃n. n ∈ ℕ ∧ 0 < n ∧ 1/n < ε
Theorem. Archimedean_property_corollary_iii
a ∈ ℝ → 0 ≤ a → (∀n. n ∈ ℕ ∧ 0 < n → a ≤ 1/n) → a = 0

Convergent sequences

Definition. converges_equi
converges a L ↔ ∀ε. ε ∈ ℝ → 0 < ε → ∃n0. n0 ∈ ℕ ∧ ∀n. n ∈ ℕ → n0 ≤ n → abs(app a n - L) < ε
Theorem. converges_fold
(∀ε. ε ∈ ℝ → 0 < ε → ∃n0. n0 ∈ ℕ ∧ ∀n. n ∈ ℕ → n0 ≤ n → abs(app a n - L) < ε) → converges a L
Theorem. converges_elim
converges a L → ε ∈ ℝ → 0 < ε → ∃n0. n0 ∈ ℕ ∧ ∀n. n ∈ ℕ → n0 ≤ n → abs(app a n - L) < ε
Theorem. const_converges
c ∈ ℝ → (∀n. n ∈ ℕ → app a n = c) → converges a c
Theorem. converges_mul_const
r ∈ ℝ → L ∈ ℝ → map a ℕ ℝ → converges a L → (∀n. n ∈ ℕ → app b n = r⋅(app a n)) → converges b (r⋅L)

Continuum induction

Theorem. continuum_induction
a ∈ ℝ → b ∈ ℝ → a < b → S ⊆ ico a b → (∀x. x ∈ ico a b → ico a x ⊆ S → ∃y. y ∈ ℝ ∧ x < y ∧ ico a y ⊆ S) → S = ico a b
