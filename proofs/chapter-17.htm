Archimedean property

Theorem. is_sup_unfold
is_sup s A → upper_bound s A ∧ (∀b. b ∈ ℝ → upper_bound b A → s ≤ b)
Theorem. upper_bound_elim
upper_bound b A → x ∈ A → x ≤ b
Theorem. nat_unbounded
¬(∃b. b ∈ ℝ ∧ upper_bound b ℕ)
Theorem. archimedean_property
x ∈ ℝ → ∃n. n ∈ ℕ ∧ x < n
Theorem. archimedean_property_corollary
x ∈ ℝ → y ∈ ℝ → 0 < x → ∃n. n ∈ ℕ ∧ y < n⋅x
Theorem. archimedean_property_corollary_ii
ε ∈ ℝ → 0 < ε → ∃n. n ∈ ℕ ∧ 0 < n ∧ 1/n < ε
