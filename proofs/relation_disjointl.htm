Theorem. relation_disjointl
R ⊆ X1 × Y1 → S ⊆ X2 × Y2 → X1 ∩ X2 = ∅ → R ∩ S = ∅
Proof
01. 1 ⊢ R ⊆ X1 × Y1, hypo.
02. 2 ⊢ S ⊆ X2 × Y2, hypo.
03. 3 ⊢ X1 ∩ X2 = ∅, hypo.
04. 3 ⊢ (X1 × Y1) ∩ (X2 × Y2) = ∅, prod_disjointl 3.
05. 1, 2 ⊢ R ∩ S ⊆ (X1 × Y1) ∩ (X2 × Y2),
  intersection_incl_in_intersection 1 2.
06. 1, 2, 3 ⊢ R ∩ S ⊆ ∅, incl_eq_trans 5 4.
07. ⊢ ∅ ⊆ R ∩ S, empty_set_is_least.
08. 1, 2, 3 ⊢ R ∩ S = ∅, incl_antisym 6 7.
relation_disjointl. ⊢ R ⊆ X1 × Y1 → S ⊆ X2 × Y2 →
  X1 ∩ X2 = ∅ → R ∩ S = ∅, subj_intro_iii 8.
