Theorem rinv_closed_positive

Theorem. rinv_closed_positive
a ∈ ℝ → 0 < a → /a ∈ ℝ
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ 0 < a, hypo.
03. 2 ⊢ ¬(0 = a), neq_from_lt 2.
04. 2 ⊢ ¬(a = 0), neq_symm 3.
05. 1, 2 ⊢ /a ∈ ℝ, rinv_closed 1 4.
rinv_closed_positive. ⊢ a ∈ ℝ → 0 < a → /a ∈ ℝ,
  subj_intro_ii 5.

Dependencies
The given proof depends on three axioms:
eq_refl, eq_subst, rinv_closed.
