Theorem. rabs_definiteness
x ∈ ℝ → abs x = 0 → x = 0
Proof
01. 1 ⊢ x ∈ ℝ, hypo.
02. 2 ⊢ abs x = 0, hypo.
03. 1 ⊢ x ≤ abs x, rabs_ge_arg 1.
04. 1, 2 ⊢ x ≤ 0, le_eq_trans 3 2.
05. 1 ⊢ -x ≤ abs x, rabs_ge_neg_arg 1.
06. 1, 2 ⊢ -x ≤ 0, le_eq_trans 5 2.
07. ⊢ 0 ∈ ℝ, calc.
08. 1 ⊢ -x ∈ ℝ, rneg_closed 1.
09. 1, 2 ⊢ -0 ≤ -(-x), rle_neg_swaps 8 7 6.
10. 1 ⊢ -(-x) = x, rneg_involution 1.
11. 1, 2 ⊢ -0 ≤ x, le_eq_trans 9 10.
12. ⊢ 0 = -0, calc.
13. 1, 2 ⊢ 0 ≤ x, eq_le_trans 12 11.
14. 1, 2 ⊢ x = 0, rle_antisym 1 7 4 13.
rabs_definiteness. ⊢ x ∈ ℝ → abs x = 0 → x = 0,
  subj_intro_ii 14.
