Theorem rmul_cancel_inv

Theorem. rmul_cancel_inv
a ∈ ℝ → b ∈ ℝ → ¬(b = 0) → a⋅b⋅/b = a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ ¬(b = 0), hypo.
04. 2, 3 ⊢ /b ∈ ℝ, rinv_closed 2 3.
05. 1, 2, 3 ⊢ a⋅b⋅/b = a⋅(b⋅/b), rmul_assoc 1 2 4.
06. 2, 3 ⊢ b⋅/b = 1, rmul_inv 2 3.
07. 2, 3 ⊢ a⋅(b⋅/b) = a⋅1, eq_cong 6, f t = a⋅t.
08. 1 ⊢ a⋅1 = a, rmul_neutr 1.
09. 1, 2, 3 ⊢ a⋅b⋅/b = a⋅1, eq_trans 5 7.
10. 1, 2, 3 ⊢ a⋅b⋅/b = a, eq_trans 9 8.
rmul_cancel_inv. ⊢ a ∈ ℝ → b ∈ ℝ →
  ¬(b = 0) → a⋅b⋅/b = a, subj_intro_iii 10.

Dependencies
The given proof depends on six axioms:
eq_refl, eq_subst, rinv_closed, rmul_assoc, rmul_inv, rmul_neutr.
