Theorem rdiv_cancel

Theorem. rdiv_cancel
a ∈ ℝ → b ∈ ℝ → ¬(b = 0) → (a/b)⋅b = a
Proof
01. 1 ⊢ a ∈ ℝ, hypo.
02. 2 ⊢ b ∈ ℝ, hypo.
03. 3 ⊢ ¬(b = 0), hypo.
04. ⊢ a/b = a⋅/b, rdiv_eq.
05. ⊢ (a/b)⋅b = (a⋅/b)⋅b, eq_cong 4, f t = t⋅b.
06. 2, 3 ⊢ /b ∈ ℝ, rinv_closed 2 3.
07. 1, 2, 3 ⊢ (a⋅/b)⋅b = a⋅(/b⋅b), rmul_assoc 1 6 2.
08. 2, 3 ⊢ /b⋅b = b⋅/b, rmul_comm 6 2.
09. 2, 3 ⊢ b⋅/b = 1, rmul_inv 2 3.
10. 2, 3 ⊢ /b⋅b = 1, eq_trans 8 9.
11. 2, 3 ⊢ a⋅(/b⋅b) = a⋅1, eq_cong 10, f t = a⋅t.
12. 1 ⊢ a⋅1 = a, rmul_neutr 1.
13. 1, 2, 3 ⊢ a⋅(/b⋅b) = a, eq_trans 11 12.
14. 1, 2, 3 ⊢ (a⋅/b)⋅b = a, eq_trans 7 13.
15. 1, 2, 3 ⊢ (a/b)⋅b = a, eq_trans 5 14.
rdiv_cancel. ⊢ a ∈ ℝ → b ∈ ℝ →
  ¬(b = 0) → (a/b)⋅b = a, subj_intro_iii 15.

Dependencies
The given proof depends on seven axioms:
eq_refl, eq_subst, rinv_closed, rmul_assoc, rmul_comm, rmul_inv, rmul_neutr.
