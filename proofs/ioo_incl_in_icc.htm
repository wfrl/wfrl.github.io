Theorem ioo_incl_in_icc

Theorem. ioo_incl_in_icc
ioo a b ⊆ icc a b
Proof
01. 1 ⊢ x ∈ ioo a b, hypo.
02. 1 ⊢ x ∈ {x | x ∈ ℝ ∧ a < x ∧ x < b},
  eq_subst ioo_eq 1, P t ↔ x ∈ t.
03. 1 ⊢ x ∈ ℝ ∧ a < x ∧ x < b, comp_elim 2.
04. 1 ⊢ x ∈ ℝ, conj_elimll 3.
05. 1 ⊢ a < x, conj_elimlr 3.
06. 1 ⊢ x < b, conj_elimr 3.
07. 1 ⊢ a ≤ x, le_from_lt 5.
08. 1 ⊢ x ≤ b, le_from_lt 6.
09. 1 ⊢ x ∈ ℝ ∧ a ≤ x, conj_intro 4 7.
10. 1 ⊢ x ∈ ℝ ∧ a ≤ x ∧ x ≤ b, conj_intro 9 8.
11. 1 ⊢ set x, set_intro 4.
12. 1 ⊢ x ∈ {x | x ∈ ℝ ∧ a ≤ x ∧ x ≤ b}, comp_intro 11 10.
13. 1 ⊢ x ∈ icc a b, eq_subst_rev icc_eq 12, P t ↔ x ∈ t.
14. ⊢ x ∈ ioo a b → x ∈ icc a b, subj_intro 13.
15. ⊢ ∀x. x ∈ ioo a b → x ∈ icc a b, uq_intro 14.
ioo_incl_in_icc. ⊢ ioo a b ⊆ icc a b, incl_intro 15.

Dependencies
The given proof depends on three axioms:
comp, eq_refl, eq_subst.
